Scrcpy Auto-Connect (Samsung S22)
Проблема: Стандартное подключение scrcpy по Wi-Fi часто обрывается при разблокировке экрана, смене разрешения или переходе в режим AOD (видеокодер Android перезапускается). Также динамический IP телефона в локальной сети делает использование статичных команд неудобным.

Решение: Отзоустойчивый Bash-скрипт, который мониторит состояние устройства и автоматически восстанавливает сессию при технических сбоях, при этом корректно завершая работу при намеренном закрытии окна пользователем.

Ключевые возможности:
Умное переподключение: Скрипт отличает ручное закрытие окна (выход) от аварийного обрыва связи (авторестарт).

Сетевое обнаружение (Discovery): Автоматический поиск IP-адреса телефона в подсети с помощью nmap, если он изменился.

Оптимизация для Samsung S22: Настроенный битрейт (8M) и параметры для стабильной передачи изображения.

Управление питанием: Автоматическое выключение экрана телефона при закрытии окна на ПК (--power-off-on-close).

Предотвращение сна: Блокировка ухода телефона в глубокий сон во время активной сессии.

Требования:
ОС: Linux (Manjaro/Arch/Ubuntu).

scrcpy версии 2.0 или выше.

android-tools (adb).

nmap (для функции автопоиска IP).

Быстрый старт:
Включите «Отладку по беспроводной сети» в параметрах разработчика.

Один раз подключите телефон кабелем и введите adb tcpip 5555.

Запустите скрипт:

Bash
chmod +x s22_wifi.sh
./s22_wifi.sh
