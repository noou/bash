# Scrcpy Auto-Connect v3.1 (Samsung Edition) 📱

**Проблема:** Падение сессии `scrcpy` при разблокировке экрана (сбой энкодера Samsung) и динамические порты Wireless Debugging.

**Решение:** Отказоустойчивый Bash-скрипт с логикой авто-обнаружения и адаптивным восстановлением связи.

### 🛠 Механика работы:
- **Fast Port Scan:** Мгновенный поиск активного порта отладки (30000-65535) без использования mDNS.
- **Smart Reconnect:** Автоматический перезапуск трансляции при сбое видеокодера Android (типично для One UI).
- **Zero-Touch:** Кэширование последнего успешного адреса для мгновенного старта.
- **Auto Power-Off:** Принудительное гашение экрана смартфона при закрытии окна на ПК (через `adb shell` + флаг `--power-off-on-close`).

### 🚀 Инструкция:
1. Включите **Wireless Debugging** в меню разработчика.
2. Скопируйте `s22_wifi.sh` в удобную папку.
3. Запустите:
   ```bash
   chmod +x s22_wifi.sh
   ./s22_wifi.sh
